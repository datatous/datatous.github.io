## 매일 사라지는 3시간 ⏳

수원에 살고 잠실로 출근한다. 편도 1시간 30분, **하루 왕복 3시간**이 지하철에서 사라진다.

처음엔 그냥 폰 보다가 내렸다. 유튜브 → 인스타 → 카톡 → 다시 유튜브. 무한 스크롤.

내릴 때쯤이면 항상 이 생각이 들었다.

> '나 지금 뭐 한 거지?'

근데 어느 날 진짜 딱 멈췄다. 이 3시간, 워크데이 기준으로만 따지면 일주일에 15시간이다. 한 달 출근일 20일이면 **60시간**. 60시간이면 작은 프로젝트 하나는 충분히 만들 수 있는 시간 아닌가?

---

## 근데 지하철에서 개발이 된다고요? 🤔

처음엔 나도 안 될 거라고 생각했다. 노트북 펴기엔 자리가 좁고, 와이파이도 끊기고, 무엇보다 화면이 너무 작다.

근데 요즘은 진짜 다르더라. 핵심은 **Claude Code**다. AI한테 자연어로 "이 코드 고쳐줘"만 말하면 알아서 짜준다. 타이핑이 줄어드니까 화면 크기는 별 상관 없어진다. 폰으로도 충분하다는 뜻이다.

---

## 처음엔 Happy 앱으로 시작했어요 📱

Claude Code는 원래 터미널에서 돌아간다. 근데 폰에는 터미널이 없다. 여기서 막혔다.

그래서 처음 찾은 게 **Happy 앱**이었다. Claude Code 세션을 폰에서 띄울 수 있게 해주는 서드파티 모바일 클라이언트인데, 집/회사 PC에 Claude Code를 설치하고 앱과 페어링하면 폰으로 "이 함수 리팩토링해줘"라고 말할 때 PC에서 Claude Code가 돌고 결과만 폰으로 받는 식이었다.

처음엔 이게 신세계였다. 지하철에서도, 카페에서도, 침대에서도 개발이 된다는 뜻이었으니까. 근데 한두 달 쓰다 보니 페어링이 끊기거나, 세션이 꼬이는 일이 종종 생겼다. 서드파티 앱이라 업데이트 주기도 들쭉날쭉했다.

---

## 지금은 Claude 자체 Remote Control로 갈아탔어요 🔄

그러다 Anthropic이 **Remote Control**이라는 기능을 공식으로 내놨다. 터미널에서 `claude --remote-control` (또는 `/remote-control`)을 실행하면 세션 URL과 QR코드가 뜨고, 그걸로 claude.ai/code나 폰의 Claude 앱에서 같은 세션에 접속할 수 있다.

가장 큰 차이는 **세션이 항상 로컬 PC에서 돈다**는 점이다. 폰이나 브라우저는 그 세션을 들여다보는 창일 뿐이고, 실제 작업은 집/회사 PC의 파일시스템과 MCP 서버, 프로젝트 설정을 그대로 쓴다. 그래서 `@`로 파일 경로를 자동완성하는 것도, 내가 깔아둔 MCP 도구를 쓰는 것도 폰에서 다 똑같이 된다.

세팅도 단순하다.

```bash
# 프로젝트 폴더에서
claude --remote-control
```

이러면 인터랙티브 세션이 시작되면서 동시에 원격 접속용 링크가 생긴다. 폰의 Claude 앱에서 'Code' 탭에 들어가면 이 세션이 컴퓨터 아이콘 + 초록 점으로 표시되고, 탭하면 바로 이어서 대화할 수 있다. PC가 잠들거나 와이파이가 끊겨도 다시 연결되면 세션이 자동으로 복구된다.

Happy 앱과 비교하면:

| | Happy 앱 | Claude Remote Control |
|---|---|---|
| 주체 | 서드파티 | Anthropic 공식 |
| 연결 방식 | 별도 페어링 | claude.ai 계정으로 바로 |
| 안정성 | 가끔 끊김 | 자동 재연결 |
| 추가 설치 | 앱 + PC 설정 | PC에서 명령어 한 줄 |

지금은 출근길에 폰으로 어제 밤에 돌려두던 작업 진행 상황을 확인하고, 필요하면 그 자리에서 다음 지시를 보낸다. PC는 집에 켜둔 채로.

## Git으로 어디서든 이어 작업 🔁

Remote Control이 "지금 떠 있는 세션"을 폰으로 들여다보는 거라면, **Git**은 "PC 자체가 꺼져 있을 때"를 위한 보험이다.

모든 작업 폴더를 GitHub에 올려놓고, 회사 PC와 집 PC에서 같은 레포를 동기화한다.

```
회사 PC에서 작업
  ↓ git push
집 PC에서 git pull → 이어 작업
  ↓ git push
다음 날 회사 PC에서 git pull → 마무리
```

처음엔 충돌도 좀 났다. 어디서 마지막에 작업했는지 헷갈리기도 했고. 근데 한 달쯤 쓰니까 몸에 익었다. 이제는 **PC 두 대 + 폰이 한 작업실처럼** 느껴진다.

---

## 실제로 뭐가 달라졌냐면 🌱

가장 큰 변화는 **부담이 사라진 거**다.

예전엔 "이거 하려면 주말에 시간 내야 하는데"였다. 지금은 "출근길에 30분만 쓰면 되겠네"가 됐다. 이 시리즈 블로그도, 자동화 프로젝트들도 대부분 지하철에서 시작됐다. 집에 와서는 막 진도 빼지 않고 **이미 80% 만들어진 걸 다듬기만** 한다.

이게 누적되니까 한 달에 한두 개씩 뭐가 만들어졌다.
