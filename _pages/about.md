## 안녕하세요, Henry입니다 👋

AX 컨설턴트로 일하면서 클라이언트한테는 "이거 자동화하면 좋아요"라고 말하면서, 정작 내 반복 업무는 손으로 하고 있다는 걸 깨달았다.

그래서 직접 만들어보기로 했다. 이 블로그는 그 과정의 기록이자, 내가 쌓아온 지식의 창고다. **만들고 싶은 걸 먼저 정하고, 필요한 기술을 그때그때 찾아 배우는 방식**으로 일한다. 코딩 실력보다 "이거 자동화되면 좋겠다"는 생각 하나가 더 중요하다고 믿는다.

## 무엇을 하는 사람인가

### 1. AX 컨설팅 & 변화관리

조직이 AI를 '도입'하는 것과 '체화'하는 것은 다르다. 교육 설계, 현장 코칭, 변화관리까지 — 기술이 실제 업무 방식의 변화로 이어지는 마지막 구간을 책임진다.

- DX/AX 변화관리 및 조직문화 컨설팅
- KT AX 디그리 2.0 — AI 전환 역량 강화 프로그램 기획·운영
- MS Power Platform 기술 컨설팅 (Power Automate, Copilot Studio, Power Apps)

### 2. 데이터 분석 & BI

3년간 공공기관에서 데이터를 다뤘다. 분석 자체보다, **분석이 의사결정으로 이어지는 구조**를 만드는 데 관심이 많다.

- 무선 네트워크 VOC 분석 → Power BI 품질 예측 대시보드 구축
- 기상 API + Python 시계열 분석으로 검사 일정 최적화
- 공공데이터 정책 연구 2년 — 과기정통부 표창 수상 (2024)

### 3. 에이전틱 시스템 & 자동화

Claude Code 위에 나만의 멀티 에이전트 시스템을 운영한다. 블로그 발행, 카드뉴스 제작, 교회 주보, 결혼 준비까지 — 반복되는 모든 것은 자동화 대상이다.

- 도메인별 워커 에이전트 + 오케스트레이터 구조 설계·운용
- LLM Wiki 기반 지식 축적 파이프라인 ([Wiki](/wiki/)에서 실시간 공개 중)
- 이 블로그의 발행 파이프라인도 그 산출물이다

**Henry's Agentic System** — Claude Code 기반 멀티 에이전트 하네스. 글쓰기·데이터 분석·MS 업데이트 추적·지식 위키 등 도메인별 워커가 협업하는 개인 자동화 인프라. 이 블로그가 그 쇼케이스다.
